About the album

a myth, a confession, a memory dressed as melody

Woven in solitude, folklore is a collection of narratives—some true, some borrowed, all haunting.

Each song, a vignette.
Each lyric, a path through the woods.
Two moments frozen like postcards—a cabin swallowed by pine and fog
a letter you never sent, stained with rose petals and regret
